## Getting Started

### Prerequisites
- Node.js (v18 or later)
- npm

### Installation

```bash
git clone https://github.com/attiquebisma8-maker/frontend-ai-capstone.git
cd frontend-ai-capstone
npm install
```

### Development

```bash
npm run dev
```

Then open the URL shown in the terminal (usually http://localhost:5173).
